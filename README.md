# Guess The Number (Python)

This is a simple **Python number guessing game** played in the terminal.

## About the Game
- The computer will generate a random number between **1 and 10**
- The user must guess the number
- The program will give hints like **Too high** or **Too low**
- It also counts the total number of guesses

## Features
- Random number generation using `random` module
- Range checking (1 to 10)
- Invalid input handling (non-numbers)
- Shows number of attempts after correct guess

## How to Run
1. Save the code in a file named:
   `Guess_the_number.py`
2. Run the program:
   ```bash
   python
   Guess_the_number.py
